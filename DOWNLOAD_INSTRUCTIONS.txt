🆘 TROUBLESHOOTING DOWNLOADS
════════════════════════════════════════════════════════════════════════════════

ISSUE: Archive won't extract
SOLUTION:
• Download again - file may be corrupted
• Try 7-Zip (https://www.7-zip.org/)
• Use command line: tar -xzf TravelKart-Complete-Project.tar.gz

ISSUE: Missing files after extraction
SOLUTION:
• Extract to empty folder
• Check if antivirus blocked files
• Try extracting again

ISSUE: npm install fails
SOLUTION:
• Make sure Node.js is installed: node --version
• Update npm: npm install -g npm@latest
• Delete node_modules and try again
• Check internet connection

ISSUE: Port 3000 or 5000 already in use
SOLUTION:
• Close other applications using these ports
• Or change port in package.json
• Use: PORT=3001 npm run dev
