## 🚀 Quick Start (5 Minutes)

```bash
# 1. Frontend
cd travelkart-frontend
npm install
npm run dev
# Opens at http://localhost:3000

# 2. Backend (in another terminal)
cd travelkart-backend
npm install
npm run dev
# Server at http://localhost:5000

# 3. Open your browser and test!
```
